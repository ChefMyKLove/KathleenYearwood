# Kathleen Yearwood

Artist website for Kathleen Yearwood — musician, composer, writer, and visual artist.

## Structure

```
index.html      — main page
styles.css      — layout and typography
bg-anim.css     — cover and moon section animations
bg-anim.js      — SVG turbulence animation driver
images/         — photos, art, and assets
```

## Sections

- **Cover** — full-screen animated night trees photo with SVG turbulence swirl
- **Photographs** — editorial band photo grid, scroll-reveal
- **Bio** — artist biography
- **The Records** — five albums with embedded Bandcamp players
- **The Art** — carousel linking to ArtPal (voiceoftheturtle)
- **Contact** — message form and mailing list
